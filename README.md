# Slim3-Starter-MVC (In development)
Slim Framework 3 starter for your new project with Twig and MVC structure

## Features

* Twig
* Eloquent
* MVC structure
* Admin panel

## Install

```shell
$ git clone https://github.com/Dezodev/Slim3-Starter-MVC.git slim
$ cd slim
$ npm install
$ php -S localhost:8000
```
